# Semaforo em Front-End
Projeto semaforo

## Descrição
Projeto de um semáforo simples usando HTML, CSS e JavaScript.  
O usuário pode ligar as luzes **vermelha**, **amarela** e **verde** clicando nos botões correspondentes.

## Funcionalidades
- Acender/vermelho, amarelo e verde.
- Botões para controlar cada luz.
- Interface simples e intuitiva.

## Como usar
1. Abra a pasta `semaforo`.
2. Abra o arquivo `index.html` no navegador.
3. Clique nos botões para ligar as luzes.

## Tecnologias
- HTML
- CSS
- JavaScript
